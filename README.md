# Tikhonov-Vladimir-Project2
Project 2. File manager with Pygame
---------------------------------------
Для использования файлового менеджера необходимо открыть файл manager.py
Для использования текстовых окон не надо на них нажимать, когда они высвечиваются.
Чтобы написать любой символ, необходимо набрать его на клавиатуре, например, при нажатии кнопки "а", в окне последнм символом добавится буква "а".
Чтобы удалить последний символ, необходимо нажать backspace, например в окне было "дом", и после нажатия на backspace, в окне будет "до".
Чтобы удалить все символы в окне, необходимо нажать delete, и тогда, весь текст в окне очистится. Например, в окне было "файл мой найден", и после нажатия на delete, а окне будет пусто.
Чтобы скопировать в окно текст, необходимо его скопировать в буфер обмена, и затем, вернувшись в менеджер, нажать на LCtrl (Левый Ctrl), и тогда текст перенесётся в текстовое окно. Например, в окне было пусто, и мы скопировали путь к папке, и нажав на LCtrl, в окне появляется путь к этой папке.
В нём откроется окно с кнопкой начать работу. После её нажатия необходимо в окно ввести путь к папке, с которой Вы хотите работать. Для этого, сначала небходимо удалить текст в окне, а затем вручную, либо скопировав, в окне написать путь. Затем подтвердить папку, нажав на RShift (Правый Shift). Например, в окне написано "user/test_folder", и после подтверждения, менеджер перенесёт на страницу этой папки.
После открытия страницы папки, на экране будет указано, какие файлы есть в папке (названия и разрешения), а также их количество.
Например:
file1.txt
file2.py
file3.txt
Количество файлов в папке: 3
Файловый менеджер позволяет копировать, удалять и искать по ключевому слову файлы.
Для копирования файла необходимо нажать на файл, а затем на другой файл с тем же разрешением. После этого, первый выбранный файл скопируется во второй. Например, мы нажимаем на file1.txt, а затем на file3.txt. После этого file3.txt становится точной копией file1.txt.
Для удаления файла необходимо нажать на файл дважды. После этого файл удалится и из папки, и из менеджера. Например, мы нажимаем на file3.txt, а звтем ещё раз на него нажимаем. После этого file3.txt удаляется.
Для поиска файлов по ключевому слову, необходимо нажать на текстовое окно под надписью "Количество файлов в папке" и ввести ключевое слово. После этого необходмо нажать на RShift (Правый Shift) и страница обновится только с фалйлами, в которых есть ключевое слово. Например, в нашей папке есть 2 файла: file1.txt и file2.py. Если ввести "2", то страница обновится только с файлом file2.py. В этом режиме нельзя использовать копирование и удаление. Чтобы вернуться к обычному режиму, необходимо нажать на текстовое окно.
